# Templa

Templa es una aplicación web progresiva de bienestar emocional orientada a acompañar al usuario en momentos de ansiedad, pausa o necesidad de calma. El proyecto propone una experiencia simple, mobile first y centrada en ejercicios de respiración, registro emocional y seguimiento del progreso personal.

## Funcionalidades implementadas

- Aplicación Web Progresiva (PWA) instalable.
- Registro e inicio de sesión de usuarios con Supabase Auth.
- Perfil de usuario autenticado.
- Edición del nombre y correo electrónico del usuario
- Cierre de sesión.
- Registro de estados de ánimo.
- Visualización de estadísticas del perfil.
- Ejercicios de respiración dinámicos.
- Botón de pausa/SOS con ejercicio guiado.
- Registro del feedback del usuario sobre la utilidad de la pausa.
- Navegación SPA con Vue Router.
- Interfaz responsive/mobile first.
- Componentización de la interfaz mediante Vue Single File Components.

## Tecnologías utilizadas

- Vue 3
- Vite
- Vue Router
- Tailwind CSS
- Supabase Auth
- Supabase Database
- JavaScript
- HTML5
- CSS3
- PWA (vite-plugin-pwa)

## Instalación

```bash
npm install
npm run dev
```
